# FlexiSpot Web Controller - ロードマップ

このドキュメントは、現行の `Vite + TypeScript` 実装を前提に、すでに着地した項目と次に進める項目を整理するためのものです。

## 現在地

v2 系では次の基盤が入っています。

- Web Serial による接続 / 切断
- UP / DOWN のホールド操作
- 4 種類のプリセット呼び出し
- 現在高さのリアルタイム表示
- セッション内の最近サンプル表示
- セッション高さチャート
- 日次高さ履歴の保存と可視化
- プリセット表示名のローカル保存
- プリセット表示名のモーダル編集
- 受信バイトの診断モニタ
- キーボードショートカット
- `protocol` / `storage` のユニットテスト
- GitHub Actions での `npm test` + `npm run build`

## 技術メモ

- 旧 `public/js/*.js` 前提の計画は破棄し、すべて `src/` 配下のモジュール単位で進める
- 高さ可視化は SVG ベースで実装済みのため、新規ライブラリ導入は必要性が出るまで見送る
- Web Serial は secure context 前提なので、PWA や配布導線の整備でも HTTPS 条件を外せない
