# FlexiSpot Web Controller - ロードマップ

このドキュメントは、現行の `Vite + TypeScript` 実装を前提に、すでに着地した項目と次に進める項目を分けて管理するためのものです。

## 現在地

v2 系では次の基盤がすでに入っています。

- Web Serial による接続 / 切断
- UP / DOWN のホールド操作
- 4 種類のプリセット呼び出し
- 現在高さのリアルタイム表示
- セッション内の最近サンプル表示
- 日次高さ履歴の保存と可視化
- プリセット表示名のローカル保存
- 受信バイトの診断モニタ
- キーボードショートカット

未実装の候補は、以下を優先順に進めます。

## 技術メモ

- 旧 `public/js/*.js` 前提の計画は破棄し、すべて `src/` 配下のモジュール単位で進める
- 高さ可視化はすでに SVG ベースで実装済みのため、新規ライブラリ導入は必要性が出るまで見送る
- Web Serial は secure context 前提なので、PWA や配布導線の整備でも HTTPS 条件を外せない
| 統計・分析 | 中 | 中 | ⭐⭐ |
| 多言語対応 | 低 | 中 | ⭐ |
| コミュニティ機能 | 低 | 高 | ⭐ |

## 🚦 実装の進め方

1. **まずは高さ表示機能から**
   - 既存のコードで高さデータを取得済み
   - UIに表示するだけなので実装が簡単
   - ユーザー価値が高い

2. **次にカスタムプリセット**
   - 既存のプリセット機能を拡張
   - LocalStorageの活用
   - ユーザーエンゲージメント向上

3. **PWA対応で利便性向上**
   - オフライン対応で信頼性向上
   - インストール可能でアクセス性向上

4. **段階的に高度な機能を追加**
   - ユーザーフィードバックを収集
   - 需要の高い機能から実装
